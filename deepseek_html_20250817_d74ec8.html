How does our system work?

Modular perception-to-narration pipeline that separates seeing from saying through structured evidence extraction.

  • YOLOv8 Static Fixtures: Detects persistent road anchors
  • EasyOCR Lexicon: Extracts verified place names
  • Optical Flow Events: Creates temporal scaffolding
  • Places365 Scene: Provides contextual priors
  • BLIP-2 Captioner: Generates grounded sentences
  • Mistral-7B Narrator: Composes constrained paragraphs
Key Innovation: External temporal scaffold + OCR-gated naming eliminates hallucinations.
